# Python Escape Room

An interactive Python terminal game where players solve puzzles and unlock doors to escape.

## Features
- Math challenge
- Pattern guessing
- Random lucky number game
- Lives system
- Reward badges

## Technologies
- Python
- Random module
- Functions
- Loops
- Conditions

## How to Run

Clone repository:

git clone <repo-url>

Run:

python "escape room project.py"
